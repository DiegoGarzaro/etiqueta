# Docs — Inventário Doméstico ("self_inventory")

A personal, responsive web app to catalog every item in the house and know exactly
where it lives — armários, gavetas, caixas and organizadores, each with a QR label.

- **Platform:** responsive web app (mobile-first PWA) · **UI language:** Português (BR)
- **Locations:** QR-code labels + photos

## Design direction in one line

**"Etiqueta"** — the identity of a labeled, orderly house. The one signature element is
the **location code shown as a physical-looking manila tag** (`ARM-A · GAV-02`), tying
every digital record to its real-world label. Warm paper surfaces, pine-green as the sole
action accent, amber reserved for attention.
